Contributing:

Our team uses Prettier and ESLint to keep the formatting of our code easy to read and consistent as well as to help catch errors in code.

  Standards:
  - Airbnb JavaScript/React style guide
  - Tabs with visual width of 2 to indent
  - single quotes
  - always use semicolons

  To Check In Code:
  1) npm run format - to fix formatting of code
  2) npm run lint - to check for coding errors
